论如何在手机上使用git

title: 论如何在手机上使用git

date: 2020-10-09 19:28:09

tags:教程

准备工具:安卓手机一台(安卓7.0以上)，utermux

ut链接，提取码mjvu

开工!

打开ut,执行以下命令: 

$ pkg install git

有y/n这种提示直接选y就可以。


安装完之后使用git --version检测一下。
现在应该可以用git clone了，但是你可能想用手机ssh连接gitee,github和gitlab。

执行命令:

$ ssh-keygen -t rsa -C youremail@youremail.com
PS:上面的命令要是生成失败就用atilo安装个debian再在debian里生成。三次回车即可生成。

执行命令cat ~/.ssh/id_rsa.pub查看公匙。

然后注册gitee,github和gitlab账号，把公匙在设置里面添加一下。

gitee,github和gitlab可以用一个公匙。

添加后，在终端输入:


#Gitee
 $ ssh -T git@gitee.com

#GitHub
 $ ssh -T git@github.com

#Gitlab
 $ ssh -T git@gitlab.com

第一次绑定的时候输入上边的代码之后会提示是否continue,输入yes后程序会自动连接，如果要求登录，直接输入登录信息即可。

再次执行上面的命令，检查是否成功连接，如果返回一下信息，则表示添加成功

You've successfully authenticated, but GitHub does not provide shell access.(github,gitee也差不多，但是gitlab的不太一样)

接下来我们要设置一些信息:

$ git config --global user.name "yourname"
$ git config --global user.email "youremail@youremail.com"

name尽量和码云或GitHub保持一致，但email必须是码云或GitHub注册时使用的邮箱。命令不分前后，没有顺序。


git使用教程自己找吧(比如https://www.cnblogs.com/yiven/p/8465054.html)
